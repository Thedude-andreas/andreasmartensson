AMC Download Manager (Android)

Enkel Android-app för att snabbt öppna senaste filer. Standard är Downloads, med möjlighet att lägga till fler mappar i en minimal mappdialog.

Syfte

  • Öppna senast nedladdade filen direkt med ett tryck.
  • Slippa leta i filsystemet efter redan nedladdade dokument.
  • Behålla ett enkelt gränssnitt utan onödiga menyer.

Funktioner

  • Start i Downloads som standard.
  • Stöd för extra mappar via enkel knapp: Mappar.
  • Sammanslagen lista, sorterad med senaste fil överst.
  • Stor tumvänlig knapp för att öppna senaste filen direkt.

Nedladdning

Ladda ner APK-paket (zip)

ZIP-paketet innehåller signerad release-APK (AMCDM-release.apk).
